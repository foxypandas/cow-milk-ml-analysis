# Прогноз продуктивности и вкуса молока у коров

Проект посвящён анализу данных о коровах и решению двух задач машинного обучения:

1. прогнозирование удоя;
2. прогнозирование вкуса молока.

В рамках проекта выполнены предобработка данных, исследовательский анализ, визуализация, обучение моделей и интерпретация результатов.

## Цель проекта

Цель проекта — выявить факторы, связанные с продуктивностью коров и вкусом молока, а также построить модели, способные предсказывать эти показатели на основе имеющихся признаков.

## Задачи проекта

### Задача 1. Прогноз удоя

Построить модель регрессии для предсказания значения `milk_yield` по характеристикам коровы и условиям содержания.

### Задача 2. Прогноз вкуса молока

Построить модель классификации для предсказания бинарного признака `taste` и определить, какие факторы сильнее всего связаны с качеством молока.
